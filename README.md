## 💻 Getting Started

To run these projects locally, you will need a web server environment that supports PHP.

### Prerequisites
- [XAMPP](https://www.apachefriends.org/index.html) or [WAMP](https://www.wampserver.com/en/) installed on your machine.

### Installation
1.  **Clone the Repository:**
    Navigate to your local server's root directory (e.g., `C:\xampp\htdocs\`) and clone the project:
    ```bash
    git clone https://github.com/ScarXiFy/CPE-3222.git
    ```
2.  **Start the Server:**
    Open the XAMPP Control Panel and start the **Apache** module.

---

## 📖 Usage

Once your local server is running, you can access the projects through your browser:

- **Resume Website:**
  Navigate to `http://localhost/CPE-3222/ResumeWebsite/index.html`
- **PHP Exercises:**
  Browse the exercises at `http://localhost/CPE-3222/phpExercises/`
